# multiChat Version 1.3.0 Releases Note

- Added icon replacement support  
You can now change the icon for newly created WeChat clones, making it easier to distinguish between different multi-instance WeChat apps.

- Added update notifications  
multiChat can now check for new versions and, when an update is available, display the release notes and a download entry point.

- Added a special marker for WeChat instances created by multiChat  
If a WeChat clone was created by multiChat, a ✨ symbol will appear before its name so you can identify it more quickly.

- Improved handling of duplicate files in the temporary folder  
If a file with the same name already exists in the temporary folder, the app now overwrites it instead of reporting an error directly, reducing interruptions to the user workflow.
